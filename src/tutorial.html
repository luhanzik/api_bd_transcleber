Guia de Integração Segura

API de Acesso ao banco_prod (Porta 3002)

🔑 Gerenciamento: Gere seus tokens no Painel Administrativo.

📊 Conexão com Power BI / Excel

Siga exatamente estes campos na tela Avançada da Web:

  1. URL: http://localhost:3002/api/NOME_DA_TABELA
  2. Cabeçalho (Header): Authorization
  3. Valor: Bearer SEU_TOKEN_AQUI

Exemplo: se o seu token for tk_123, o valor deve ser Bearer tk_123

Importante: Se o Power BI pedir o tipo de credencial depois, selecione Anônimo.

💻 Integração em Sistemas

Header HTTP obrigatório:

Authorization: Bearer [TOKEN]

🔗 Atalhos

  • Painel Admin (Gerar Tokens)
  • Swagger (Testar Online)
